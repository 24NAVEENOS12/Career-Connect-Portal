# 📖 Project Overview

**Career Connect Portal** is a complete recruitment platform developed using **PHP**, **MySQL**, **Bootstrap**, **HTML**, **CSS**, and **JavaScript**.

The system provides a dedicated portal for **Candidates** and **Companies**, enabling the complete recruitment lifecycle—from registration and job posting to candidate selection.

The project focuses on:

* Modern User Interface
* Secure Authentication
* Resume Management
* Recruitment Workflow
* Responsive Design
* Easy Job Management
* Professional Dashboard Analytics

# 🎯 Why Career Connect Portal?

Many traditional recruitment systems suffer from:

* ❌ Complex application process
* ❌ Poor communication between recruiters and candidates
* ❌ Difficult applicant tracking
* ❌ Lack of centralized recruitment management

### Career Connect Portal solves these problems by providing:

✅ Centralized Recruitment Platform             ✅ Secure Candidate & Company Authentication
✅ Easy Job Posting & Management                ✅ Professional Candidate Profiles
✅ Resume Upload & Download                     ✅ Recruitment Status Tracking
✅ Dashboard Analytics                          ✅ Responsive Web Application

# ⚙ Installation Guide

Follow these steps to run the project on your local machine.

## 1️⃣ Clone the Repository
```bash
git clone https://github.com/24NAVEENOS12/Career-Connect-Portal.git
```

## 2️⃣ Move Project to XAMPP

Copy the project folder into:
```text
xampp/
   └── htdocs/
         └── Career-Connect-Portal/
```

## 3️⃣ Start XAMPP

Open **XAMPP Control Panel**

Start the following services:

* ✅ Apache
* ✅ MySQL

## 4️⃣ Create Database

Open
```
http://localhost/phpmyadmin
```

Create a database named
```text
careerconnectportal
```

## 5️⃣ Import SQL File

Click
```
Import
```

Choose
```
careerconnectportal.sql
```

Click
```
Go
```

## 6️⃣ Configure Database Connection

Open
```
config/database.php
```

Update
```php
$host = "localhost";
$user = "root";
$password = "";
$database = "careerconnectportal";
```

## 7️⃣ Run the Project

Open your browser
```
http://localhost/Career-Connect-Portal/
```

🎉 Your application is now ready.
